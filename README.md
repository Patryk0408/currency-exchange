# [Currency exchange](https://patryk0408.github.io/currency-exchange/)

## Description

Task of this web application is currency exchange.  
Thanks to this, you can find out how much your money is worth.

> If you want to exchange your money, you must enter data to the form.

## Technology

I used Java Script technology and BEM.  
Thanks to BEM, code is more transparent and more understandable.
